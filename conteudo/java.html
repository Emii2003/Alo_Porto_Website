Java

Nesta disciplina, a ideia é utilizar da programação orientada a objetos com java para desenvolver uma aplicação para o obtenção de dados e atualização de dados referentes aos dados do cliente, guincho, caminhões, etc. Existem cálculos a serem feitos para determinar o melhor guincho com base na tensão comparada ao peso do veículo pesado do cliente. E portanto, esses cálculos deverão ser feitos em Python, mas também colocar essa opção em Java é uma viabilidade.
